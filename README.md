Simulates multiple tasks running simultaneously using animated progress bars in the terminal.

<h2>Features</h2>
<ul>
<li>Multiple independent progress bars <br> </li>
<li>Randomized task speeds <br> </li>
<li>Colored terminal output <br> </li>
<li>Real-time screen refresh animation <br> </li>
<li>Percentage tracking <br> </li>
<li>Automatic completion detection </li>
</ul>
